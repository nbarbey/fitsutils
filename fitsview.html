fitsutils by nbarbey

A set of small command-line tools for FITS files

fitsview

Why fitsview

fitsview allows you to have a quick look at a fits image using matplotlib.

A simple exemple

Get a sample fits file: 
wget http://www.cv.nrao.edu/fits/data/samples/image/swp05569slg.fits
To display it on screen, simply type: 
fitsview swp05569slg.fits

Installation

If you do not want to install fitsutils and still want to use fitsview, you can simply put the fitsview file in your path. Do not forget to make it executable: 
wget https://github.com/nbarbey/fitsutils/raw/master/fitsutils/fitsview.py
chmod +x fitsview.py
get the source code on GitHub : nbarbey/fitsutils
