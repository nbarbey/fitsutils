fitsutils by nbarbey

A set of small command-line tools for FITS files

fitsupdate

Why fitsupdate

fitsupdate is a simple command-line tool to update the value associated with a keyword in a give fits file header. Its aims at being simple and flexible.

A simple exemple

As in the fitsheader exemples you can get a sample file like this: 
wget http://www.cv.nrao.edu/fits/data/samples/image/swp05569slg.fits
Assume you want to update the DATE record of the primary header to the current date. You can type : 
fitsupdate -e 0 -r DATE -v "`date`" swp05569slg.fits
The -v option set the value the DATE record will be updated with. The "`date`" input will evaluate the date command and output the results. The double quotes are necessary to handle the whitespaces in the output of date. To check that everything went fine, you can use fitsheader: 
fitsheader -e 0 -r DATE swp05569slg.fits
If you do not want to install fitsutils and still want to use fitsupdate, you can simply put the fitsheader file in your path. Do not forget to make it executable: 
wget https://github.com/nbarbey/fitsutils/raw/master/fitsutils/fitsupdate.py
chmod +x fitsupdate.py
