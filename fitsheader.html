fitsutils by nbarbey

A set of small command-line tools for FITS files

fitsheader

Why fitsheader

Reading a fits header from the command-line interface can be a real headache. Indeed, the primary fits header is stored as text at the beginning of the fits file. So you can see it with a simple more command. But then it is stored as a very long single line so it will look really bad if your terminal is not exactly 80 character wide. And what if you want to look at an extended fits header ? You just can't.

But those tasks are very easy in Python thanks to PyFITS. So why not write a simple script which will internally call PyFITS and nicely display the output in the terminal ? That's exactly what fitsheader is doing.

A simple exemple

FITS archive at NRAO provides people with reference articles about the FITS standard and also provides nice FITS file exemples . So let us grab an exemple file and see what we get with fitsheader: 
wget http://www.cv.nrao.edu/fits/data/samples/image/swp05569slg.fits
fitsheader swp05569slg.fits

Let us assume you are only interested in the first extension. Just type: 
fitsheader -e 1 swp05569slg.fits
and you will get what you want. Note that the extension number is not provided here as you asked for a specific extension number. 
fitsheader -r NAXIS swp05569slg.fits
will give you the number of dimensions for each extension 
fitsheader -e 0 -r BITPIX swp05569slg.fits
will give you the float encoding of the extension number 0. You can display headers of multiple files at once with. The optional argument -n will display the name of the file at the beginning of each header: 
fitsheader -n *
A great thing about fitsheader is that you can combine it with the power of the command line. If you want to list the keywords of the first extension of a FITS file, you can just : 
fitsheader -e 1 swp05569slg.fits | cut -f 1
or to get only the values : 
fitsheader -e 1 swp05569slg.fits | cut -f 2
You can store an header in a text file : 
fitsheader swp05569slg.fits >> header.txt
You get the idea. If you do not want to install fitsutils and still want to use fitsheader, you can simply put the fitsheader file in your path. Do not forget to make it executable. 
wget https://github.com/nbarbey/fitsutils/raw/master/fitsutils/fitsheader.py
chmod +x fitsheader.py
