fitsutils by nbarbey

A set of small command-line tools for FITS files

fitsfilter

Why fitsfilter

fitsfilter allows you to filter a set of fits file according to values in their header.

A simple exemple

Get a sample fits file: 
wget http://www.cv.nrao.edu/fits/data/samples/image/swp05569slg.fits
The syntax of fitsfilter is a bit odd but allows a lot of flexibility in the tests. To start simple, let us filter files with two-dimensional arrays in the primary extension. 
fitsfilter -t a[NAXIS] == 2 *
To distinguish between record names and values, one need to specify the key in a[] and the value alone. For equality test, do not forget the == condition is required. A single = will not be recognized. Any test working under Python should work with fitsfilter, so you can perform inequality tests. 
fitsfilter -t a[NAXIS] >= 2 *

Installation

If you do not want to install fitsutils and still want to use fitsfilter, you can simply put the fitsfilter file in your path. Do not forget to make it executable: 
wget https://github.com/nbarbey/fitsutils/raw/master/fitsutils/fitsfilter.py
chmod +x fitsfilter.py
get the source code on GitHub : nbarbey/fitsutils
